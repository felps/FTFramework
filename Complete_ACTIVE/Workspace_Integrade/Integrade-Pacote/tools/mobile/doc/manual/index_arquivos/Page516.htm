             O GridProxy também possui um arquivo de configuração. Segue a lista de atributos:

instalação

 

    Proxy.properties

Atributos

Descrição

server.address

Endereço através do qual o Proxy vai se conectar a aplicação servidora

proxy.server.ip_address

Endereço através do qual o Proxy aguarda conexões de clientes

proxy.server_tcp.port

Porta onde o Proxy espera pela conexão TCP de aplicações clientes

proxy.client_tcp.port

Porta onde o Proxy espera pela conexão TCP de aplicações servidoras

proxy.public_local.ip_address

 

 

    GridProxy.properties

Atributos

Descrição

gridproxy.port

Porta onde o Grid espera por requisições do proxyAdapter

proxyadapter.port

Porta onde o proxyAdapter espera por respostas do GridProxy

gridproxy.database 

Será TRUE, caso o banco de dados H2 seja usado

proxyadapter.host

Endereço  onde o Proxy está executando
