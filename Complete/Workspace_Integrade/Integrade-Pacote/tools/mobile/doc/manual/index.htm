Manual do Mobile integrade

Início

O Mobile Integrade é uma  infra-estrutura de software para acesso aos serviços do middleware de grade Integrade por parte de usuários de dispositivos móveis.

 

Através desta, clientes móveis podem:

¨ Solicitar a execução de aplicações na grade;

¨ Realizar o acompanhamento da execução das aplicações;

¨ Visualizar o resultado de computações já concluídas.

 

 Para tanto, a arquitetura proposta possui mecanismos de ciência de contexto, provendo suporte a períodos de desconexão e baixa conectividade bem como à adaptação de conteúdo dos resultados das computações realizadas pela grade.
