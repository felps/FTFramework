A interface Gráfica MASCT depende de uma máquina virtual JAVA com configuração CDC 1.0 e personal profile 1.0. Para rodar  o aplicativo MASCT, o usuário deve carregar para o dispositivo as dependências acima relacionadas  e o arquivo masct.jar . Após a execução do aplicativo, algumas configurações devem ser especificadas através de uma aba na própria interface (Configurations): o endereço MAC do dispositivo, o endereço do Proxy, a porta do Proxy e a porta do MASCT. Pode-se ainda, indicar um intervalo de dias para recuperação do status das aplicações que foram submetidas a grade baseado na data em que foram submetidas. 

Para captura as informações de contexto do dispositivo móvel o usuário deve instalar e executar o componente Monitor da arquitetura MoCA.
